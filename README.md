# Suzerain Modding Kit

A WIP API for creating Suzerain mods with MelonLoader.

Suzerain Modding Kit is an unofficial community-made toolkit for modding Suzerain. It is not affiliated with, endorsed by, or sponsored by Torpor Games.

## Supported platforms

- **Windows** — native, the original target.
- **Linux** — supported via Steam Proton. Suzerain itself ships only as a Windows binary, so the mod runs inside Proton's Wine prefix; the build toolchain and developer workflow are fully native (no Wine required to build).
- **macOS** — should work via Whisky/CrossOver under the same model as Linux. Untested by the maintainers; bug reports welcome.

See the [installation guide](docs/guides/user/installing-mods.md) for setup instructions on each platform.
